# Network Management - Examples

**Version:** 1.0.0.0-beta

Usage examples for Network Management.

## Basic Usage

### Scan network for devices

```
/network scan
```


### List all devices

```
/network devices
```


### Wake device with WOL

```
/network wol
```



## Advanced Usage

See [Commands Reference](COMMANDS.md) for all available commands.
