# Automation - Commands

**Version:** 1.0.0.0-beta

Complete command reference for Automation.

## Commands

### `/automation schedule`

Manage scheduled tasks

**Usage:**
```
/automation schedule
```

**Permissions:** User

---


### `/automation devicetrigger`

Device-based automation triggers

**Usage:**
```
/automation devicetrigger
```

**Permissions:** User

---


### `/automation health`

Device health monitoring

**Usage:**
```
/automation health
```

**Permissions:** User

---


### `/automation speedalert`

Internet speed alerts

**Usage:**
```
/automation speedalert
```

**Permissions:** User
