# Automation - Examples

**Version:** 1.0.0.0-beta

Usage examples for Automation.

## Basic Usage

### Manage scheduled tasks

```
/automation schedule
```


### Device-based automation triggers

```
/automation devicetrigger
```


### Device health monitoring

```
/automation health
```



## Advanced Usage

See [Commands Reference](COMMANDS.md) for all available commands.

## Tips

- Check command help with `/help`
- Use autocomplete for easier input
- Commands are case-insensitive

## Troubleshooting

If you encounter issues:
1. Check bot permissions
2. Verify command syntax
3. Check bot logs
4. Open an issue on GitHub
