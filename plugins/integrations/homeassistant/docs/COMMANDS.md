# Home Assistant - Commands

**Version:** 1.0.0.0-beta

Complete command reference for Home Assistant.

## Commands

### `/homeassistant lights`

List all lights

**Usage:**
```
/homeassistant lights
```

**Permissions:** User

---


### `/homeassistant light`

Control a light

**Usage:**
```
/homeassistant light
```

**Permissions:** User

---


### `/homeassistant switches`

List all switches

**Usage:**
```
/homeassistant switches
```

**Permissions:** User

---


### `/homeassistant switch`

Control a switch

**Usage:**
```
/homeassistant switch
```

**Permissions:** User

---


### `/homeassistant sensors`

List all sensors

**Usage:**
```
/homeassistant sensors

**Permissions:** User

---


### `/homeassistant sensor`

**Usage:**
```
