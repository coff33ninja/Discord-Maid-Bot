# Power Management - Commands

**Version:** 1.0.0.0-beta

Complete command reference for Power Management.

## Commands

### `/power control wake`

Wake device using WOL

**Usage:**
```
/power control wake
```

**Permissions:** User

---


### `/power control shutdown`

Shutdown device remotely

**Usage:**
```
/power control shutdown
```

**Permissions:** User

---


### `/power control restart`

Restart device remotely

**Usage:**
```
/power control restart
```

**Permissions:** User

---


### `/power control status`

Check device power status

**Usage:**
```
/power control status
```

**Permissions:** User

---


### `/power control configure`

Configure device for remote power control

**Usage:**
```
/power control configure
```

**Permissions:** User

---
