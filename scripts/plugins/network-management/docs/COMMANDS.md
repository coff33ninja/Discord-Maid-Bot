# Network Management - Commands

**Version:** 1.0.0.0-beta

Complete command reference for Network Management.

## Commands

### `/network scan`

Scan network for devices

**Usage:**
```
/network scan
```

**Permissions:** User

---


### `/network devices`

List all devices

**Usage:**
```
/network devices
```

**Permissions:** User

---


### `/network wol`

Wake device with WOL

**Usage:**
```
/network wol
```

**Permissions:** User

---


### `/device config`

Configure device

**Usage:**
```
/device config
```

**Permissions:** User

---


### `/device group`

Manage device groups

**Usage:**
```
/device group
```

**Permissions:** User

---
