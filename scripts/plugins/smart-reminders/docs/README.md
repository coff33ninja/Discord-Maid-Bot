# Smart Reminders

**Version:** 1.0.0.0-beta  
**Category:** Utility  
**Author:** Discord Maid Bot Team

Context-aware reminder system with time and presence-based triggers

## Features

- Time-based reminders
- Recurring reminders
- Presence-based triggers
- AI-generated variations

## Commands

- `/bot reminder` - Manage reminders

## Installation

This plugin is included by default.

## Configuration

No additional configuration required.
