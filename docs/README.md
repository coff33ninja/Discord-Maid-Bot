## Quick Start

1. Clone the repository
2. Copy `.env.example` to `.env` and fill in your tokens
3. Run `npm install`
4. Run `npm start`

## Requirements

- Node.js 18+
- Discord Bot Token
- Gemini API Key (for AI features)
- Optional: Home Assistant, Tailscale, SMB share
