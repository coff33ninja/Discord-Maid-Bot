## üöÄ Quick Start

```bash
# 1. Clone repository
git clone https://github.com/coff33ninja/Discord-Maid-Bot.git
cd Discord-Maid-Bot

# 2. Install dependencies
npm install

# 3. Configure environment
cp .env.example .env
# Edit .env with your tokens

# 4. Start bot
npm start
```
